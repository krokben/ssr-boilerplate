# SSR Boilerplate

Based on https://github.com/thomasrogerlux/typescript-react-ssr-boilerplate.

To install:

```
npm install
```

To run locally:

```
npm run build:dev
```

then

```
npm run start:dev
```

To run production:

```
npm run build
```

then

```
npm run start
```

App is available by default on port `3000`.
